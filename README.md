MailNest is email client built with React and Firebase. It provides a clean, intuitive interface for managing your emails with features like authentication, mail composition, inbox management, and more.

## Features

- **User Authentication**: Secure login, signup, and password recovery using Firebase Authentication
- **Mail Management**: View, compose, and organize emails in Inbox, Sent, and Starred folders
- **Rich Text Composition**: Compose emails with rich text formatting using React Quill
- **Responsive Design**: Mobile-friendly interface built with Tailwind CSS
- **State Management**: Efficient state handling with Redux Toolkit

## Tech Stack

- **Frontend**: React
- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit, React Redux
- **Routing**: React Router v7
- **Backend**: Firebase (Authentication, Firestore)
